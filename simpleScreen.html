Programa de captura de tela no Linux: Simple Screen Recorder

Passo 1: Você pode baixar ou instalar o Simple Screen Recorder, de acordo com a sua distribuição linux, que pode ser observado na tabela abaixo.

Arch Linux Basta instalar 'simplescreenrecorder':
sudo pacman -S simplescreenrecorder
Adicione se quiser gravar aplicativos OpenGL de 32 bits em um sistema de 64 bits:
sudo pacman -S lib32-simplescreenrecorder
Debian Se você estiver usando o Debian 9 ou mais recente, você pode instalar o pacote 'simplescreenrecorder' dos repositórios oficiais. Você pode instalá-lo com:
sudo apt-get update
sudo apt-get install simplescreenrecorder
Adicione se quiser gravar aplicativos OpenGL de 32 bits em um sistema de 64 bits:
sudo apt-get install simplescreenrecorder-lib: i386
Você deve copiar e colar esses comandos em um terminal linha por linha, em vez de todos de uma vez, caso contrário, não funcionará.
Fedora SimpleScreenRecorder está disponível no repositório RPM Fusion. O RPM Fusion pode ser ativado com este comando:
rpm -Uvh http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-stable.noarch.rpm
Depois disso, SimpleScreenRecorder pode ser instalado com este comando:
dnf install simplescreenrecorder
Gentoo SimpleScreenRecorder agora está incluído no repositório principal. Você pode instalá-lo com:
sudo emerge -av media-video / simplescreenrecorder
KaOS SimpleScreenRecorder está disponível no repositório oficial, o nome do pacote é 'simplescreenrecorder'.
Mageia Instale simplescreenrecorder do repositório Core Release usando urpmi ou o Centro de Controle Mageia.
NixOS Você pode instalar SimpleScreenRecorder com o seguinte comando:
nix-env -i simplescreenrecorder
O mesmo comando também deve funcionar em qualquer outra distribuição, desde que o gerenciador de pacotes Nix esteja instalado.
OpenSUSE Instale ' simplescreenrecorder ' do repositório Packman.
Se você deseja gravar aplicativos OpenGL de 32 bits em um sistema de 64 bits, você também deve instalar 'libssr-glinject-32bit'.
Este pacote foi criado por Dmitriy (DAP-DarkneSS).
Red Hat Enterprise Linux , CentOS , Scientific Linux Os pacotes para SimpleScreenRecorder estão disponíveis nos repositórios do Nux.
Slackware Um SlackBuild está disponível aqui.
Ubuntu , Linux Mint , ... Se você estiver usando o Ubuntu 17.04 ou mais recente, SimpleScreenRecorder pode ser encontrado nos repositórios oficiais. Você pode instalá-lo com:
sudo apt-get update
sudo apt-get install simplescreenrecorder
Você deve copiar e colar esses comandos em um terminal linha por linha, em vez de todos de uma vez, caso contrário, não funcionará.
Adicione se quiser gravar aplicativos OpenGL de 32 bits em um sistema de 64 bits:
sudo apt-get install simplescreenrecorder-lib: i386
Se você estiver usando uma versão mais antiga do Ubuntu ou quiser obter a versão mais recente do SimpleScreenRecorder imediatamente, sem esperar pelo próximo lançamento do Ubuntu, você também pode obter o (s) pacote (s) no SimpleScreenRecorder PPA:
sudo apt-add-repository ppa: maarten-baert / simplescreenrecorder
sudo apt-get update
sudo apt-get install simplescreenrecorder
Adicione se quiser gravar aplicativos OpenGL de 32 bits em um sistema de 64 bits: sudo apt-get install simplescreenrecorder-lib: i386
Outras distribuições Linux Para todas as outras distribuições, você deve compilá-lo por enquanto. Não é tão difícil e há instruções no arquivo leia-me. Você pode encontrar o código-fonte no GitHub:
https://github.com/MaartenBaert/ssr
Se você não estiver familiarizado com o git e quiser apenas um arquivo .tar.gz, baixe este:
https://github.com/ MaartenBaert / ssr / archive / master.tar.gz
